Audio Files for Spotify Clone

Place your audio files in this directory.

Required file:
- Future-technology.mp3 (main demo track for the player)

You can:
1. Download a royalty-free track and rename it to "Future-technology.mp3"
2. Create your own audio file with this name
3. Use a text-to-speech service to create a sample audio file

Without this file, the player will show an error when you try to play music.

For testing, you can find royalty-free music at:
- FreePD.com
- Pixabay.com/music
- SoundCloud (with Creative Commons licenses)
